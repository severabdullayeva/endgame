# Endgame

Endgame is a React-based frontend project built with Vite.  
It was developed as a practice project to improve skills in React, JavaScript, and component-based UI development.

---

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3
- Git & GitHub

---

## Features

- Responsive user interface
- Component-based architecture
- Interactive UI elements
- State management using React Hooks
- Clean and reusable code structure

---
